
# Lucent.AI – Static Export (Squarespace/Vercel Ready)

## What’s inside
- HTML: index, about, playbook, assessment, success, contact
- CSS: css/style.css
- Images: images/LucentAI Icon.png, images/LucentAI Logo 2.png
- robots.txt, sitemap.xml

## Squarespace (Developer Mode)
1) Open Website → Developer Mode → File Manager.
2) Upload everything from this zip to the root.
3) Create pages and set their URL slugs to match (/, /about, /playbook, /assessment, /success, /contact).
4) For Assessment, keep the embedded ScoreApp script (already included).

## Vercel
1) Create new project → “Import project” → drag & drop the unzipped folder.
2) Set “Framework” to “Other” (plain static). Deploy.
